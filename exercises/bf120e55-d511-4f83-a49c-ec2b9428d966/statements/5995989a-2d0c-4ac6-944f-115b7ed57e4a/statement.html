Utilizando la base de datos world, realiza la siguiente consulta

Utiliza la cláusula EXCEPT para mostrar los países de la región 'Southern Europe' de los que conocemos su headofstate y no tienen una forma de gobierno de 'Constitutional Monarchy'.

Ordena los resultados por el nombre de los países.

Solución del ejercicio (solo para pruebas)

lenguaje:

PostgreSQL

solución


select name
from country
where headofstate is not null and region = 'Southern Europe' 
    except
select name from country where governmentform = 'Constitutional Monarchy'
order by name;

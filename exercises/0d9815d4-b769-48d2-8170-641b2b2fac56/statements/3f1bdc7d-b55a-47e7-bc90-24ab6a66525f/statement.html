En este ejercicio, vamos a suponer que tenemos correctamente creada la tabla farmacias. Se debe crea una tabla llamada farmaceuticos.

La tabla debe tener los siguientes atributos:

  • un id autoincremental que actúe como clave primaria,
  • un nombre de hasta 100 caracteres de longitud,
  • una dirección de hasta 200 caracteres de longitud,

Posteriormente, añadiremos a la tabla farmacias un atributo farmaceutico_id, así como una restricción de clave ajena que relacionen el campo farmaceutico_id con la tabla farmaceuticos, de tal forma que si un farmaceutico se elimina, se deben eliminar todas las farmacias relacionadas con el anterior.
